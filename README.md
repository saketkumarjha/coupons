# Coupon Scraper Service

Automated scraper for e-commerce coupon sites with intelligent keyword extraction and categorization.

## Features
- Scrapes CouponDunia, GrabOn, and CashKaro
- Smart keyword extraction using NLP
- Automatic categorization
- Stores in Supabase
- Runs every 6 hours via cron
